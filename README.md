ENG

Develop a data warehouse using SQL Server that includes ETL processes, data modelling and analytics. 
This project consolidates customer, product and sales data and enables analytical reporting and supports data-driven decision-making.
1. Build the Data Warehouse
Objective: Build a data warehouse using SQL Server that includes ETL processes, data modelling
Specifications
Data Sources:		Import data from source systems (ERP and CRM) provided as CSV files     
Data Quality:		Clean and resolve data quality issues before analysis.
Integration:	  Combine sources into a single data model for analytical queries.
Scope:			    Focus on the latest dataset only; historization of data is not required.
Documentation:	Provide clear documentation of the data model to support both business stakeholders and analytics requests.	

2. BI: Analytics and Reporting
Objective: Develop SQL-based analytics to deliver detailed insights about customer behaviour, product performance and sales trends

FR

Concevoir un entrepôt de données sous SQL Server intégrant des processus ETL, la modélisation et l'analyse des données . Ce projet consolide les données clients, produits et ventes, permet la production de rapports analytiques et facilite la prise de décision basée sur les données.

1. Création de l'entrepôt de données
Objectif : Créer un entrepôt de données sous SQL Server incluant les processus ETL et la modélisation des données.
Spécifications
Sources de données : Importer les données des systèmes sources (ERP et CRM) fournies sous forme de fichiers CSV.
Qualité des données : Nettoyer et résoudre les problèmes de qualité des données avant l'analyse.
Intégration : Combiner les sources en un modèle de données unique pour les requêtes analytiques.
Périmètre : Se concentrer uniquement sur le jeu de données le plus récent ; l'historisation des données n'est pas requise
Documentation : Fournir une documentation claire du modèle de données afin de répondre aux besoins des parties prenantes et aux demandes d'analyse.

2. BI: Analyse et reporting
Objectif : Développer des analyses basées sur SQL pour fournir des informations détaillées sur le comportement des clients, les performances des produits et les tendances des ventes.
